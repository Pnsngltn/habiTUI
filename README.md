# habiTUI

### This is (going to be) **habiTUI**.

A simple terminal-based habit tracker with heatmap calendar visualization.

## Installation

```bash
pip install textual
```

## Usage

python main.py

## Controls

- Arrow keys/vim keys: navigate
- Enter: select/confirm
- Space: mark complete
- q: quit

## Features

- Add/delete habits
- Daily/weekly streak tracking
- Heatmap calendar view

_Designed to help you keep your objectives in sight._
